桌面App

桌面App让您更轻松地管理 etcd，并且非常轻量只有 5M，支持 Mac 和 Windows 的主流版本。

Download

Web版部署

支持部署私有Web服务，用户无需额外下载就可以在任意设备上通过浏览器管理etcd。并为Windows、MacOS和Linux（arm / x86）提供Docker镜像，你可以在Docker中直接启动它。

Docker Hub

键值编辑器

提供高亮代码编辑器，支持多种语言的高亮和快速代码格式化，对kubernetes protobuf做了特殊的支持，让你管理Key-Value更加得心应手。

历史版本对比

支持查看并对比 Key 不同修订版本的内容，当更新Key时发生冲突有友好的界面进行内容合并处理。

Key监听通知

提供Key创建、修改、删除的实时事件推送，即使在后台运行也能收到通知。
